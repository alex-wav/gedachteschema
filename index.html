Google Drive koppelen

Voer eenmalig je Google OAuth Client ID in om schema's direct op te slaan in je Drive.

  1. Ga naar console.cloud.google.com
  2. Maak een project aan en activeer Google Drive API
  3. Credentials → Create → OAuth 2.0 Client ID
  4. Type: Web application — voeg je GitHub Pages URL toe als authorized origin
  5. Kopieer de Client ID en plak hieronder
